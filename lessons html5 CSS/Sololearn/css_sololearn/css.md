–ü—Ä–µ–¥—Å—Ç–∞–≤–ª–µ–Ω–∏–µ —á–∏—Å–µ–ª –≤ —Å–∏—Å—Ç–µ–º–µ **HEX** —è–≤–ª—è–µ—Ç—Å—è –±–æ–ª–µ–µ —ç—Ñ—Ñ–µ–∫—Ç–∏–≤–Ω—ã–º. –ù–∞ —Ü–≤–µ—Ç —Ç—Ä–µ–±—É—é—Ç—Å—è —Ç–æ–ª—å–∫–æ 2 —Ü–∏—Ñ—Ä—ã. –¶–≤–µ—Ç–æ–≤—ã–µ –º–æ–¥–µ–ª–∏ –∏ –∫–æ–¥—ã –Ω—É–∂–Ω—ã, –ø–æ—Ç–æ–º—É —á—Ç–æ —Å—É—â–µ—Å—Ç–≤—É–µ—Ç —Å–ª–∏—à–∫–æ–º –º–Ω–æ–≥–æ —Ü–≤–µ—Ç–æ–≤, —á—Ç–æ–±—ã –∫–∞–∂–¥–æ–º—É –¥–∞—Ç—å –æ—Ç–¥–µ–ª—å–Ω–æ–µ –∏–º—è.

–£—Å—Ç—Ä–æ–π—Å—Ç–≤–∞ –æ—Ç–æ–±—Ä–∞–∂–µ–Ω–∏—è, —Ç–∞–∫–∏–µ –∫–∞–∫ —ç–∫—Ä–∞–Ω —Ç–µ–ª–µ—Ñ–æ–Ω–∞, –º–æ–Ω–∏—Ç–æ—Ä–∞, —Ç–µ–ª–µ–≤–∏–∑–æ—Ä–∞, —Å–æ—Å—Ç–æ—è—Ç –∏–∑ –∫—Ä–æ—à–µ—á–Ω—ã—Ö —ç–ª–µ–º–µ–Ω—Ç–æ–≤ –∏–∑–æ–±—Ä–∞–∂–µ–Ω–∏—è, –∏–∑–≤–µ—Å—Ç–Ω—ã—Ö –∫–∞–∫ –ø–∏–∫—Å–µ–ª–∏. –ö–∞–∂–¥—ã–π –ø–∏–∫—Å–µ–ª—å –º–æ–∂–µ—Ç –∏–∑–ª—É—á–∞—Ç—å –∫–æ–º–±–∏–Ω–∞—Ü–∏—é *–∫—Ä–∞—Å–Ω–æ–≥–æ*, *–∑–µ–ª–µ–Ω–æ–≥–æ* –∏ *—Å–∏–Ω–µ–≥–æ* —Å–≤–µ—Ç–∞ –¥–ª—è —Å–æ–∑–¥–∞–Ω–∏—è –ª—é–±–æ–≥–æ —Ü–≤–µ—Ç–∞ –≤ *–º–æ–¥–µ–ª–∏ RGB*.

- **–ø—Ä–æ—Ü–µ–Ω—Ç—ã** `%` –ø–æ–∑–≤–æ–ª—è—é—Ç —Å–æ–∑–¥–∞–≤–∞—Ç—å –±–æ–ª–µ–µ –∞–¥–∞–ø—Ç–∏–≤–Ω—ã–µ –¥–∏–∑–∞–π–Ω—ã, –∫–æ—Ä—Ä–µ–∫—Ç–∏—Ä—É—è—Å—å –≤ –∑–∞–≤–∏—Å–∏–º–æ—Å—Ç–∏ –æ—Ç —Ä–∞–∑–º–µ—Ä–æ–≤ —Ä–æ–¥–∏—Ç–µ–ª—å—Å–∫–∏—Ö —ç–ª–µ–º–µ–Ω—Ç–æ–≤

–ß—Ç–æ–±—ã —Å–æ–∑–¥–∞–≤–∞—Ç—å —Å—Ç—Ä–∞–Ω–∏—Ü—ã, –∫–æ—Ç–æ—Ä—ã–µ –≤—ã–≥–ª—è–¥—è—Ç –∫—Ä–∞—Å–∏–≤–æ –∏ —Å–±–∞–ª–∞–Ω—Å–∏—Ä–æ–≤–∞–Ω–æ, –Ω—É–∂–Ω–æ –ø–æ–ª–Ω–æ—Å—Ç—å—é –∫–æ–Ω—Ç—Ä–æ–ª–∏—Ä–æ–≤–∞—Ç—å —Ä–∞–∑–º–µ—Ä—ã —ç–ª–µ–º–µ–Ω—Ç–æ–≤. –î–∏—Å–ø–ª–µ–π, –ø–æ–¥–æ–±–Ω—ã–π —ç–∫—Ä–∞–Ω—É —Ç–µ–ª–µ—Ñ–æ–Ω–∞, —Å–æ—Å—Ç–æ–∏—Ç –∏–∑ —ç–ª–µ–º–µ–Ω—Ç–æ–≤ –∏–∑–æ–±—Ä–∞–∂–µ–Ω–∏—è –∏–ª–∏ –ø–∏–∫—Å–µ–ª–µ–π. –ü–∏–∫—Å–µ–ª–∏ –∏—Å–ø–æ–ª—å–∑—É—é—Ç—Å—è –≤ –∫–∞—á–µ—Å—Ç–≤–µ **–µ–¥–∏–Ω–∏—Ü—ã –∏–∑–º–µ—Ä–µ–Ω–∏—è** –¥–ª—è –∫–æ–Ω—Ç—Ä–æ–ª—è —Ä–∞–∑–º–µ—Ä–æ–≤ —ç–ª–µ–º–µ–Ω—Ç–æ–≤. –°–≤–æ–π—Å—Ç–≤–æ `font-size` –∫–æ–Ω—Ç—Ä–æ–ª–∏—Ä—É–µ—Ç —Ä–∞–∑–º–µ—Ä —Ç–µ–∫—Å—Ç–∞:

–í –±–æ–ª—å—à–∏–Ω—Å—Ç–≤–µ –≤–µ–±-–±—Ä–∞—É–∑–µ—Ä–æ–≤, **—Ä–∞–∑–º–µ—Ä —Ç–µ–∫—Å—Ç–∞** –≤ –∞–±–∑–∞—Ü–∞—Ö, –∫–Ω–æ–ø–∫–∞—Ö, —Å–ø–∏—Å–∫–∞—Ö –∏ —Ç–∞–±–ª–∏—Ü–∞—Ö **–ø–æ —É–º–æ–ª—á–∞–Ω–∏—é —Ä–∞–≤–µ–Ω** `16px`. –ú–æ–∂–Ω–æ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å –∑–Ω–∞—á–µ–Ω–∏—è, —Ç–∞–∫–∏–µ –∫–∞–∫ `small (13px)`, `medium (16px)` –∏ `large (18px)` –¥–ª—è —Ä–∞–∑–º–µ—Ä–∞ —Ç–µ–∫—Å—Ç–∞. –£ –Ω–∏—Ö —É–∂–µ —É—Å—Ç–∞–Ω–æ–≤–ª–µ–Ω—ã –µ–¥–∏–Ω–∏—Ü—ã –ø–∏–∫—Å–µ–ª–µ–π.

–°–æ–≤—Ä–µ–º–µ–Ω–Ω—ã–µ –≤–µ–±-—Å–∞–π—Ç—ã —Å–æ–∑–¥–∞—é—Ç—Å—è —Ç–∞–∫–∏–º –æ–±—Ä–∞–∑–æ–º, —á—Ç–æ–±—ã –≤—ã–≥–ª—è–¥–µ—Ç—å –æ—Ç–ª–∏—á–Ω–æ –Ω–∞ —ç–∫—Ä–∞–Ω–∞—Ö —Ä–∞–∑–ª–∏—á–Ω—ã—Ö —Ä–∞–∑–º–µ—Ä–æ–≤ –∏ —Ä–∞–∑—Ä–µ—à–µ–Ω–∏–π. –≠—Ç–æ—Ç –ø–æ–¥—Ö–æ–¥ –∏–∑–≤–µ—Å—Ç–µ–Ω –∫–∞–∫ –∞–¥–∞–ø—Ç–∏–≤–Ω—ã–π –≤–µ–±-–¥–∏–∑–∞–π–Ω. –ò—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏–µ –ø–∏–∫—Å–µ–ª–µ–π –≤ –∫–∞—á–µ—Å—Ç–≤–µ –µ–¥–∏–Ω–∏—Ü—ã –∏–∑–º–µ—Ä–µ–Ω–∏—è –Ω–µ —è–≤–ª—è–µ—Ç—Å—è –ª—É—á—à–µ–π –ø—Ä–∞–∫—Ç–∏–∫–æ–π –≤ –æ—Ç–Ω–æ—à–µ–Ω–∏–∏ –∞–¥–∞–ø—Ç–∏–≤–Ω–æ–≥–æ –≤–µ–±-–¥–∏–∑–∞–π–Ω–∞. –≠–∫—Ä–∞–Ω—ã —Ä–∞–∑–Ω—ã—Ö —Ä–∞–∑–º–µ—Ä–æ–≤ –∏ —Ä–∞–∑—Ä–µ—à–µ–Ω–∏–π –∏–º–µ—é—Ç —Ä–∞–∑–Ω–æ–µ –∫–æ–ª–∏—á–µ—Å—Ç–≤–æ –ø–∏–∫—Å–µ–ª–µ–π, –∞ –ø–∏–∫—Å–µ–ª–∏ —è–≤–ª—è—é—Ç—Å—è –µ–¥–∏–Ω–∏—Ü–∞–º–∏ —Ñ–∏–∫—Å–∏—Ä–æ–≤–∞–Ω–Ω–æ–≥–æ —Ä–∞–∑–º–µ—Ä–∞.

–ß—Ç–æ–±—ã –¥–∏–∑–∞–π–Ω –≤–∞—à–µ–π —Å—Ç—Ä–∞–Ω–∏—Ü—ã –æ—Ç–ª–∏—á–Ω–æ –≤—ã–≥–ª—è–¥–µ–ª –Ω–∞ –ª—é–±–æ–º —ç–∫—Ä–∞–Ω–µ, –∏—Å–ø–æ–ª—å–∑—É—é—Ç—Å—è **–ø—Ä–æ—Ü–µ–Ω—Ç—ã** (`%`) –¥–ª—è —É—Å—Ç–∞–Ω–æ–≤–∫–∏ –æ—Ç–Ω–æ—Å–∏—Ç–µ–ª—å–Ω—ã—Ö —Ä–∞–∑–º–µ—Ä–æ–≤. –≠—Ç–æ –æ–±–µ—Å–ø–µ—á–∏–≤–∞–µ—Ç –ø–æ—Å–ª–µ–¥–æ–≤–∞—Ç–µ–ª—å–Ω—ã–π –¥–∏–∑–∞–π–Ω –Ω–∞ —Ä–∞–∑–Ω—ã—Ö —Ä–∞–∑–º–µ—Ä–∞—Ö —ç–∫—Ä–∞–Ω–∞. –ü—Ä–∏ —Å—Ç–∏–ª–∏–∑–∞—Ü–∏–∏ —ç–ª–µ–º–µ–Ω—Ç–∞ —Å –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏–µ–º –ø—Ä–æ—Ü–µ–Ω—Ç–æ–≤ (%), 100% –±—É–¥–µ—Ç —Å–æ–æ—Ç–≤–µ—Ç—Å—Ç–≤–æ–≤–∞—Ç—å –ø–æ–ª–Ω–æ–º—É —Ä–∞–∑–º–µ—Ä—É –µ–≥–æ —Ä–æ–¥–∏—Ç–µ–ª—å—Å–∫–æ–≥–æ —ç–ª–µ–º–µ–Ω—Ç–∞.
