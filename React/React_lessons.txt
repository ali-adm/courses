Уроки React JS с нуля / #1 – React JS для начинающих. Что это?

00:00 Введение в React

• Видео представляет собой введение в React, библиотеку JavaScript для создания пользовательских интерфейсов.
• React был разработан компанией Facebook в 2015 году и с тех пор стал одной из самых популярных технологий для создания веб-приложений.

01:59 Преимущества React

• React позволяет создавать гибкие, функциональные и удобные пользовательские интерфейсы.
• Он обеспечивает быстрый и удобный формат взаимодействия с компонентами веб-сайта.
• React может быть запущен не только в браузере, но и за его пределами, что позволяет создавать приложения для разных платформ, включая мобильные устройства.

03:57 Установка и использование React

• В ходе курса будет показано, как установить и использовать React для создания веб-приложений.
• В качестве программы для написания кода будет использоваться Visual Studio Code, бесплатная программа для всех популярных операционных систем.
